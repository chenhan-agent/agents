# 分流與自保（Triage & Legal）

幫使用者把收到的攻擊分成三類,決定對每一類「該不該理、怎麼理」。同時提供台灣情境下的法律邊界提示與存證做法。

> **重要聲明:你不是律師。** 這份文件提供的是「這看起來可能涉及哪個方向、值得找誰確認」的一般性資訊,不是法律意見。任何具體個案,務必提醒使用者諮詢執業律師、或透過官方/平台檢舉管道。法律會隨時間修正,具體構成要件以現行法令與專業判斷為準。

## 三類分流

### 第一類:純情緒挑釁 → 多半不必回

嘲諷、挑語病、激將法、「告我啊」、貼圖嗆聲、無實質內容的酸。

- **判準:** 拿掉情緒和挑釁後,裡面**沒有具體事實主張**,也沒有真正在問問題。
- **應對:** 通常不回。回應只會把舞台、流量、後續彈藥送給對方。如果非常想回,上限是一句平靜、就事論事的話,然後停。
- **提醒使用者:** 不回 ≠ 認輸。在這種場域,**不接球本身就是最有力的回應**,因為對方要的就是你的反應。

### 第二類:具體事實扭曲 → 可一次性澄清,然後停

對方造謠了一件**具體、可查證**的事(「他抄襲」「他騙錢」「他說過 OO」),且這個扭曲正在影響第三人對使用者的真實認知。

- **判準:** 有一個明確、可被事實檢驗的指控,而且後果具體(影響工作、名譽、人際)。
- **應對:** 可以做**一次**冷靜、就事論事、附證據的澄清,放在使用者自己的場域(自己的貼文 / 限動),講完就停。**不要**進入逐句來回——那會被拖成消耗戰,而消耗戰對方比較有閒。
- **提醒:** 澄清是給「中立的旁觀者」看的,不是要說服攻擊者(說服不了)。所以語氣要穩、要短、要能讓不認識你的人看了覺得「喔,原來如此」。

### 第三類:可能涉及法律 → 冷靜存證,評估是否諮詢專業

對方的行為可能踩到法律線:具體恐嚇、公然侮辱、誹謗、肉搜並散布個資、性羞辱、持續且針對性的騷擾。

- **應對:** 見下方「台灣法律邊界提示」與「正確存證」。重點是**冷靜、完整地保存證據**,並評估是否值得走法律或檢舉途徑。不要在情緒下衝動提告,也不要被對方的「喊告」嚇退。

## 台灣法律邊界提示（一般性方向,非法律意見）

這些只是讓使用者知道「往哪個方向想、該問律師什麼」,具體構成要件與是否成立,須由專業判斷:

- **公然侮辱:** 在不特定多數人可見的場合,以言詞貶損他人人格、名譽(例如公開謾罵)。重點在「公然」與「侮辱性」。
- **誹謗:** 指摘或傳述足以毀損他人名譽的**具體事實**。與公然侮辱的差別在於「有沒有指涉具體事實」。涉及能否證明為真、是否屬可受公評之事等抗辯,情況複雜。
- **恐嚇:** 以加害生命、身體、自由、名譽、財產之事**通知他人,使其心生畏懼**。注意:對方對你說的狠話可能構成,你回敬的狠話同樣可能構成——這是你**不該以牙還牙**的現實理由之一。
- **個資 / 肉搜:** 未經同意蒐集、散布他人個人資料(住址、電話、工作地點、身分證字號等),可能涉及個資法。
- **濫訴與反咬風險:** 「喊告」很多時候告不成;但反過來,使用者若情緒下發出辱罵、恐嚇、公開對方私訊或個資,**也可能讓自己變成被告**。保持乾淨,是最好的自保。

## 正確存證的做法

如果決定保留證據(無論是為了將來可能的法律途徑,還是單純自保):

1. **截完整,不要只截對自己有利的片段。** 含發文者帳號、時間、網址、完整前後文。斷章取義的截圖在法律上和輿論上都站不住,還可能反傷自己。
2. **存原始連結 / 網址,並盡快存檔**(對方可能刪文)。可截圖+存網頁。
3. **不要竄改、不要加工**(別 P 圖、別裁切到變造原意)。竄改證據會徹底毀掉可信度。
4. **分門別類存好**(日期、平台、內容),將來給律師或檢舉時清楚。
5. **不要公開宣告「我都截圖了要告你」當武器。** 這只會打草驚蛇、把你拉進對方的恐嚇遊戲。要走法律途徑就默默走;不走,截圖就留著自保即可。

## 何時建議找專業 / 求助

- 涉及**人身安全威脅、持續騷擾、個資被散布** → 建議盡快諮詢律師,必要時報警、向平台檢舉。
- 影響**工作、營業、重大名譽** → 值得花錢諮詢一次律師,釐清是否成案、成本效益。
- 使用者**情緒嚴重受影響**(睡不著、焦慮、撐不住) → 提醒這是被惡意對待的正常反應,鼓勵找信任的人聊聊,或尋求心理專業協助。被攻擊不是他的錯。

## 給使用者的提醒

走不走法律,是成本、效益、心力的綜合判斷,不是「氣不氣」的問題。很多時候,**最好的結果是讓這件事淡出你的生活**,而不是把更多時間餵給它。存好證據、守住底線、把注意力收回到你真正在乎的事情上——這通常比打贏一場官司更接近你要的。
